# KuroLabs Shell Aesthetic Design

Everything in KuroLabs that is not vertical-specific should have a sleek, minimal design with a paper-style feel — not clean white, but slightly beige/yellowish. Monospace typography throughout, with everything in lowercase.

This applies to all shared shell pages, which currently includes:

- Login page
- Hub page

**Design direction:**
- Color: paper/beige tones (not pure white)
- Typography: monospace font
- Case: lowercase
- Overall feel: minimal and sleek
