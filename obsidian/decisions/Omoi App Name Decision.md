# Omoi App Name Decision

The thought capture vertical will be named **Omoi** (想い / 思い), the Japanese word for "thought" or "feeling."

**Why:** It pairs naturally with Kioku — Kioku holds memory of the outside world (bookmarks, links), while Omoi holds thoughts from the inside. Together they form a coherent brand story across two verticals.

**How to apply:** Use "Omoi" as the app name in all code, branding, API routes, package names, and documentation going forward.
