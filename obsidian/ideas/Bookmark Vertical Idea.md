# Bookmark Vertical Idea

A future vertical inspired by a previous app called **Memento**, focused on saving bookmarks. Key features and principles:

## API
- Create, list, and get bookmarks

## Chrome Web Extension
- Easily create a new bookmark from the current page
- Fetches the full HTML of the page and converts it to Markdown (strips all HTML, keeps readable content)
- If the page contains a YouTube video, embed the video reference into the bookmark's Markdown

## Mobile App
- Share target for Android and iOS — any content shared to the app creates a new bookmark
- Follows the same principle: convert to Markdown, strip HTML

## Naming Ideas
The vertical should have a Japanese-inspired name. Some suggestions:

- **Tobira** (扉) — "door/gateway", each bookmark is a door to something saved
- **Shirushi** (印) — "mark/sign", as in a bookmark mark
- **Kioku** (記憶) — "memory", a fitting nod to the predecessor app Memento
- **Tameru** (溜める) — "to collect/accumulate"
- **Tsumugi** (紡ぎ) — "to weave/spin", weaving together saved content
