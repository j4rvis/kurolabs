# Per-Vertical Theme on Login

Each vertical should have its own theme. When a user opens the web app for a specific vertical directly (e.g. Shiren), the login page should use that vertical's theme rather than the default KuroLabs shell aesthetic.

**Example:** Accessing the Shiren app → login page shows the Shiren theme.

This is a future task — theming per vertical is not yet implemented.
